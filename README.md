# Diyar El Izz

A static portfolio website for ديار العز للتطوير العقاري.

## Overview

- Arabic RTL layout
- Company branding with logo
- About section with company image
- WhatsApp contact integration

## Files

- `index.html` — main website page
- `styles/` — CSS files for layout, RTL support, animations, and variables
- `assets/images/logo/logo.svg` — company logo asset

## Usage

Open `index.html` in a browser or serve the folder with any static server.
